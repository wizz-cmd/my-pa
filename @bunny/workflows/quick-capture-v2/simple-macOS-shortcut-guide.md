## Installation (5 Minutes)

### Step 1: Open Shortcuts App

1. Open **Shortcuts** app on Mac
2. Click **"+"** (New Shortcut)
3. Name it: **"Voice Note to Ingest"**

### Step 2: Add Actions (Only 4!)

#### Action 1: Dictate Text
- Search: **"Dictate Text"**
- Settings:
  - Language: **German & English**
  - Stop Listening: **After Pause**

#### Action 2: Text (Create YAML Header)
- Search: **"Text"**
- Content:
```yaml
---
type: voice-note
timestamp:
device: Mac
---


```

- After the last `---`, add a new line, then:
  - Click **Variables** ‚Üí Select **"Dictated Text"** from Action 1

#### Action 3: Save File
- Search: **"Save File"**
- Settings:
  - **Ask Where to Save:** OFF
  - **Destination:** Click "Choose" ‚Üí Navigate to:
    ```
    ~/localvault/Chris Notes/Ingest/voice-notes/
    ```
  - **File Name:** Click in field, then:
    - Type: `voice-`
    - Variables ‚Üí **Current Date** (Format: Custom `yyyyMMdd-HHmmss`)
  - **Overwrite if exists:** ON

#### Action 4: Show Notification
- Search: **"Show Notification"**
- Title: **"Voice note captured üé§"**
- Body: **"Ready for Conny to process"**

### Step 3: Add Keyboard Shortcut

1. Click the shortcut name dropdown at top
2. Select **"Add Keyboard Shortcut"**
3. Press: **‚åò‚å•V**
4. Click **"Add"**

### Step 4: Test!

## How Processing Works

### Step 1: You Create Voice Note
Press **‚åò‚å•V** ‚Üí Speak ‚Üí Done!

### Step 2: Conny Processes
- Manually: "Conny, process ingest folder"
- Automatically: Conny checks periodically

### Step 3: Conny Does the Magic
1. Reads file from `Ingest/voice-notes/`
2. Detects entities (PRE21, Roche, etc.)
3. Creates smart Wikilinks
4. Generates unique ID
5. Formats journal entry
6. Appends to `000-JOURNAL.md`
7. Moves processed file to `Ingest/processed/YYYY-MM/`

### Step 4: You See Result in Journal
```markdown
### [INFO-20260112-014] [Voice Note]
**Zeit:** 2026-01-12 23:15
**Typ:** Information

Test voice note for new ingest architecture

**Backlinks:** (auto-generated by Conny)
---
```

## Troubleshooting

### Issue: "Can't save file"
**Solution:** Verify path exists:
```bash
ls ~/localvault/Chris\ Notes/Ingest/voice-notes/
```
If not found, create it:
```bash
mkdir -p ~/localvault/Chris\ Notes/Ingest/voice-notes/
```

### Issue: "Dictation not working"
**Solution:** System Settings ‚Üí Privacy & Security ‚Üí Microphone ‚Üí Enable for Shortcuts

### Issue: "Files not being processed"
**Solution:** Tell Conny: "Process ingest folder"
Or wait for automatic processing (if enabled)
